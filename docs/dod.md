# Definition of Done (DOD)

1. Código implementado, com testes feitos pelo próprio desenvolvedor;
1. Código submetido por pull request;
1. Código revisado por um colega;
1. Foram feitos testes para garantir que nada do que já foi feito foi comprometido;
1. Funcionalidade seguiu os padrões de estilização estabelecido;
1. Garantir que o código seguiu as políticas de segurança da informação/software;
1. Documentação atualizada, se necessário.
