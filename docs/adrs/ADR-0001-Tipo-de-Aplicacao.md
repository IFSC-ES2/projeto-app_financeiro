# ADR-0001 — Tipo de Aplicação

## Status
Em espera

## Contexto
Precisa ser definido o tipo de aplicação o projeto será.

## Decisão
A equipe decidiu que o projeto será primeiramente uma aplicação web.


## Alternativas consideradas
- Aplicação desktop
- Aplicativo mobile

## Consequências
### Positivas
- Consolidação de aprendizado já desenvolvido do grupo
- Sem necessidade de colocar a aplicação em alguma loja de aplicativos

### Negativas / trade-offs
- Estar preso a ter o acesso de clientes somente em navegadores
- Necessidade de uma possível compra de dominio e plano de hospedagem
- Uma aplicação web acaba sendo menos atrativo para nosso público, que talvez prefirisse uma aplicação em seu dispositivo móvel.

## Revisão futura
A decisão poderá ser revista caso a aplicação apresente muitos problemas no ambiente web e caso o grupo entenda que um outro modelo seja melhor
